## 2. 기술 스택 (React 전환)

### Frontend
```
React 18 + TypeScript
Vite 5 (빌드 — GitHub Pages 배포 유지 or Vercel 이전)
Tailwind CSS + 자체 Design Token (행성별 컬러 시스템)
Framer Motion (게임 애니메이션)
Zustand (전역 상태 — 학생 세션, 진행도)
React Query (데이터 페칭)
```

### 게임 엔진
```
단순 게임 (카드, 매칭): 순수 React + CSS
물리 게임 (미로, 낙하, 퍼즐): React + Matter.js
음악/리듬 게임: Web Audio API
복잡한 애니메이션: Phaser.js 3 (iframe 임베드)
```

### 백엔드 (Phase 2)
```
Supabase
  - Auth (교사 계정, 학생 프로필)
  - PostgreSQL (학습 기록, 뱃지, 출석)
  - Storage (교사 업로드 사진)
  - Realtime (멀티플레이 게임)

Toss Payments (구독 결제)
Vercel (배포 + Edge Functions)
```

### 모니터링
```
PostHog (게임 이벤트 분석 — 완료율, 이탈 지점)
Sentry (에러 추적)
```

## 5. 디자인 시스템

### 일러스트 방향
- **스타일**: 플랫 벡터 + 두꺼운 아웃라인 (Duolingo, Khan Academy Kids 참고)
- **캐릭터**: 5종 고정 마스코트 (곰, 토끼, 여우, 부엉이, 강아지) — 모든 게임에 일관 등장
- **색상**: 행성별 컬러 시스템 (현재 hex 그대로 유지)
- **폰트**: Nunito + Noto Sans KR (현재 유지)
- **아이콘**: 직접 SVG 제작 또는 Phosphor Icons 커스텀
- **사운드**: 효과음 (정답/오답/완료) + 배경음 (2~3종 선택 가능)

### 피해야 할 것
- AI 이미지 느낌 (현재 문제)
- 이모지 의존
- 과도한 그라디언트
- 복잡한 레이아웃

### 참고할 앱
- Khan Academy Kids (캐릭터 중심)
- Duolingo (보상 피드백)
- Endless Alphabet (언어)
- Thinkrolls (사고력)
