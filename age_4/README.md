# Age 4 Game Plan

## Age Focus
- 만 4세반
- `기억`, `패턴`, `위치`, `집중력` 요소를 넣을 수 있다

## UI Rules
- 게임 규칙은 1개 중심으로 유지
- 카드 수는 늘릴 수 있지만 화면 복잡도는 낮게 유지
- 텍스트 설명은 짧게

## Recommended Games

### 1. 기본 이미지 매칭
- 규칙:
  - 카드 공개 후 뒤집고 같은 그림을 맞춘다
- 필요한 것:
  - 카테고리 이미지 또는 이모지
- `image_matching`에서 가져올 것:
  - 거의 전부 재사용 가능

### 2. 순서 기억 게임
- 규칙:
  - 순서대로 나온 이미지를 같은 순서로 누른다
- 필요한 것:
  - 순서 강조 효과
  - 단계별 길이 증가
- `image_matching`에서 가져올 것:
  - 상태 관리
  - 타이머/효과음

### 3. 패턴 완성 게임
- 규칙:
  - 반복 규칙을 보고 다음 이미지를 고른다
- 필요한 것:
  - 반복 가능한 패턴 데이터

### 4. 위치 기억 게임
- 규칙:
  - 물건의 위치를 잠깐 보여준 후 기억해서 맞춘다
- 필요한 것:
  - 큰 위치 슬롯
  - 짧은 공개 단계

### 5. 다른 그림 찾기
- 규칙:
  - 두 그림의 다른 점을 찾는다
- 필요한 것:
  - 미세 차이 이미지 쌍

### 6. 교실 물건 찾기
- 규칙:
  - 교실 배경에서 지정한 물건을 찾는다
- 필요한 것:
  - 교실 배경
  - 물건 오브젝트

### 7. 반대 규칙 게임
- 규칙:
  - 사과가 나오면 바나나를 누르고
  - 해가 나오면 달을 누르는 식으로
  - 평소와 반대로 선택해야 한다
- 필요한 것:
  - 반대쌍 이미지 세트
  - 짧은 라운드 구조
- 구현 메모:
  - 규칙은 한 번에 1개만 보여주는 것이 좋다

## Build Rule
- `image_matching`이 이 연령대 핵심 레퍼런스
- 결과 화면, 기록, 오디오 구조를 재사용 가능
- 카드 게임 외에도 공개 단계 -> 플레이 단계 구조를 활용할 수 있다

## First Build Priority
1. 기본 이미지 매칭
2. 순서 기억 게임
3. 위치 기억 게임
4. 반대 규칙 게임

## Needed Assets
- 카테고리별 이미지
- 패턴 데이터
- 위치 슬롯 UI
- 교실 일러스트
