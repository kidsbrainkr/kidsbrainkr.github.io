# Age Game Additions

기존 연령별 게임에 신규 1종씩 추가. 기준: 두뇌발달 · 직관성 · 웹 구현 난이도 · 기존 게임과의 중복 최소화.

## 추가 게임 목록

| 연령 | 게임명 | 발달 포인트 |
|------|--------|------------|
| age_2 | 색깔 바구니 넣기 | 색 분류, 손-눈 협응, 단순 규칙 이해 |
| age_3 | 짝이 되는 집 찾기 | 연관 개념 형성, 분류, 기초 문제 해결 |
| age_4 | 반대 규칙 게임 | 억제 조절, 규칙 유지, 작업기억 |
| age_5 | 규칙 바꾸기 분류 게임 | 인지 유연성, 규칙 전환, 주의 조절 |
| mixed | 리듬 따라하기 게임 | 작업기억, 청각-운동 통합, 집중력 |

## 게임별 핵심 규칙

**age_2 색깔 바구니 넣기**
- 큰 물건 2~3개 → 같은 색 바구니로 넣기
- UI: 오브젝트 매우 크게, 선택지 2개부터, 텍스트 최소화
- 구현: 클릭형 또는 드래그형 (클릭 후 자동 이동도 가능)

**age_3 짝이 되는 집 찾기**
- 동물/물건을 보고 맞는 집/장소 찾기 (물고기→물, 새→둥지, 자동차→차고)
- UI: 이미지 중심, 선택지 3~4개, 정답 피드백 빠르고 단순

**age_4 반대 규칙 게임**
- 사과가 나오면 바나나를 누르는 반대 선택
- UI: 규칙 한 번에 1개, 시각적으로 분명한 반대쌍
- 메모: go/no-go 이전 단계로도 적합

**age_5 규칙 바꾸기 분류 게임**
- 라운드1: 색깔 기준 분류 → 라운드2: 모양 기준 분류 (같은 카드, 다른 곳)
- UI: 현재 규칙 상단에 크게 표시, 규칙 변경 시 애니메이션/소리
- 메모: 논문 기반으로 가치 높은 유형

**mixed 리듬 따라하기 게임**
- 빛/아이콘 순서 보고 같은 순서로 누르기
- 난이도: 쉬움(2~3개) / 보통(4~5개) / 어려움(6개↑)
- UI: 버튼 크게, 속도 느리게 시작

## 우선순위
1. age_5 규칙 바꾸기 분류 게임
2. mixed 리듬 따라하기 게임
3. age_4 반대 규칙 게임
4. age_3 짝이 되는 집 찾기
5. age_2 색깔 바구니 넣기
