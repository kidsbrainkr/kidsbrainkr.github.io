# Age Game Additions

## Purpose
- 기존 연령별 게임 계획을 덮어쓰지 않고, 각 연령 그룹에 신규 게임 1개씩만 추가하기 위한 보완 문서다.
- 기준은 `두뇌발달`, `직관성`, `웹 구현 난이도`, `기존 게임과의 중복 최소화`다.

## Added Games Summary

### age_2 추가 게임
- 게임명: `색깔 바구니 넣기`
- 핵심 규칙:
  - 화면에 큰 물건 2~3개가 보인다
  - 아래에 큰 색깔 바구니가 있다
  - 같은 색 바구니로 물건을 넣는다
- 발달 포인트:
  - 색 분류
  - 손-눈 협응
  - 단순 규칙 이해
- UI 원칙:
  - 오브젝트 매우 크게
  - 선택지 2개부터 시작
  - 텍스트 최소화
- 구현 메모:
  - 클릭형 또는 드래그형 가능
  - 드래그가 부담되면 클릭 후 자동 이동 구조도 가능

### age_3 추가 게임
- 게임명: `짝이 되는 집 찾기`
- 핵심 규칙:
  - 동물이나 물건을 보고 맞는 집 또는 장소를 찾는다
  - 예:
    - 물고기 -> 물
    - 새 -> 둥지
    - 자동차 -> 차고
- 발달 포인트:
  - 연관 개념 형성
  - 분류
  - 기초 문제 해결
- UI 원칙:
  - 이미지 중심
  - 선택지는 3~4개 이하
  - 정답 피드백은 빠르고 단순하게

### age_4 추가 게임
- 게임명: `반대 규칙 게임`
- 핵심 규칙:
  - 사과가 나오면 바나나를 누르고
  - 해가 나오면 달을 누르는 식으로
  - 평소와 반대로 선택해야 한다
- 발달 포인트:
  - 억제 조절
  - 규칙 유지
  - 작업기억
- UI 원칙:
  - 규칙은 한 번에 1개만
  - 시각적으로 분명한 반대쌍 사용
- 구현 메모:
  - go/no-go 이전 단계로도 적합

### age_5 추가 게임
- 게임명: `규칙 바꾸기 분류 게임`
- 핵심 규칙:
  - 처음에는 색깔 기준으로 분류
  - 다음 라운드에서는 모양 기준으로 분류
  - 같은 카드라도 기준이 바뀌면 다른 곳으로 가야 한다
- 발달 포인트:
  - 인지 유연성
  - 규칙 전환
  - 주의 조절
- UI 원칙:
  - 현재 규칙을 상단에 크게 표시
  - 규칙이 바뀌는 순간 애니메이션 또는 소리로 알려주기
- 구현 메모:
  - 논문 기반으로도 가치가 높은 유형

### mixed 추가 게임
- 게임명: `리듬 따라하기 게임`
- 핵심 규칙:
  - 빛 또는 아이콘이 정해진 순서로 나온다
  - 같은 순서로 누르거나 두드린다
- 발달 포인트:
  - 작업기억
  - 청각-운동 통합
  - 집중력
- 난이도 확장:
  - 쉬움: 2~3개 순서
  - 보통: 4~5개 순서
  - 어려움: 6개 이상
- UI 원칙:
  - 버튼 크게
  - 속도는 느리게 시작
  - 성공 시 리듬 효과 강화

## Recommendation
- 위 5개는 기존 목록과 크게 겹치지 않으면서도 두뇌발달 측면에서 확장 가치가 높다
- 우선순위는 아래가 적절하다:
  1. age_5 `규칙 바꾸기 분류 게임`
  2. mixed `리듬 따라하기 게임`
  3. age_4 `반대 규칙 게임`
  4. age_3 `짝이 되는 집 찾기`
  5. age_2 `색깔 바구니 넣기`
