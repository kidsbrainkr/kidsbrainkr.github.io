# Age 3 Game Plan

## Age Focus
- 만 3세반
- `같은 것 찾기`, `분류`, `쉬운 짝맞추기`가 적합하다

## UI Rules
- 설정 항목은 매우 적게 유지
- 선택지는 4~6개 이하
- 카드 수는 적고 이미지 크기는 크게 유지
- 텍스트보다 아이콘과 그림 중심

## Recommended Games

### 1. 쉬운 이미지 매칭
- 규칙:
  - 4쌍 또는 5쌍 정도의 쉬운 카드 짝맞추기
- 필요한 것:
  - 큰 카드
  - 긴 공개 시간
- `image_matching`에서 가져올 것:
  - 카드 렌더링
  - 공개 후 뒤집기
  - 매칭 판정

### 2. 그림자 맞추기
- 규칙:
  - 물건과 그림자를 연결한다
- 필요한 것:
  - 그림 이미지
  - 그림자 이미지
- 구현 메모:
  - 카드 뒤집기 없이 1:1 연결형도 가능

### 3. 모양 짝맞추기
- 규칙:
  - 같은 도형을 찾는다
- 필요한 것:
  - 원, 세모, 네모, 별 등

### 4. 동물 먹이 연결하기
- 규칙:
  - 동물과 맞는 먹이를 연결한다
- 필요한 것:
  - 동물 이미지
  - 먹이 이미지

### 5. 숨은 물건 찾기
- 규칙:
  - 간단한 화면 안에서 지정한 물건 하나 찾기
- 필요한 것:
  - 배경 이미지
  - 물건 오브젝트

## Build Rule
- 기록 저장은 선택사항
- 오답 패널티는 짧고 가볍게
- 카드 수는 적게, 클릭 영역은 크게

## First Build Priority
1. 쉬운 이미지 매칭
2. 그림자 맞추기

## Needed Assets
- 주제별 이미지 세트
- 쉬운 배경 일러스트
- 짧은 성공/실패 효과음
