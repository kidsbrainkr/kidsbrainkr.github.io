# Mixed Age Game Plan

## Age Focus
- 혼합반 또는 넓은 연령대를 함께 커버해야 할 때 사용
- 같은 게임 안에서 난이도 조절이 가능한 구조가 중요하다

## UI Rules
- 기본 규칙은 하나로 유지
- 난이도만 바꿔 연령 차이를 흡수
- 카테고리/난이도 선택은 짧고 단순하게

## Recommended Games

### 1. 난이도 선택형 이미지 매칭
- 규칙:
  - 같은 게임에서 카드 수와 공개 시간을 조정
- 필요한 것:
  - 쉬움/보통/어려움 프리셋
- `image_matching`에서 가져올 것:
  - 카드 수
  - 공개 시간
  - 카테고리 선택

### 2. 단계 선택형 순서 기억 게임
- 규칙:
  - 3단계, 4단계, 5단계 순서 기억
- 필요한 것:
  - 난이도별 시퀀스 길이

### 3. 쉬운/어려운 숨은 그림 찾기
- 규칙:
  - 같은 게임에서 물건 수만 조절
- 필요한 것:
  - 단계별 배경

### 4. 친구 맞춤 모드
- 규칙:
  - 쉬움: 얼굴-얼굴
  - 어려움: 얼굴-이름
- 필요한 것:
  - 친구 데이터

## Build Rule
- 한 게임을 여러 연령대에 맞추는 방식으로 설계
- 설정이 많아질 수 있으니 프리셋 버튼으로 압축

## First Build Priority
1. 난이도 선택형 이미지 매칭
2. 친구 맞춤 모드

## Needed Assets
- 공통 이미지 세트
- 난이도 프리셋 데이터
- 단계별 문구
