# Age 5 Game Plan

## Age Focus
- 만 5세반
- `이름`, `규칙`, `기억`, `학기 초기 적응`이 핵심이다

## UI Rules
- 텍스트 사용 가능
- 다만 여전히 큰 카드와 큰 버튼이 유리함
- 설정은 있어도 되지만 한 화면에 너무 많이 두지 않는다

## Recommended Games

### 1. 친구 얼굴 이름 매칭
- 규칙:
  - 얼굴 카드와 이름 카드를 짝으로 맞춘다
- 필요한 것:
  - 사진 업로드
  - 이름 입력
  - 저장 모드 선택
- `image_matching`에서 가져올 것:
  - friend mode를 그대로 기반으로 사용
- 중요:
  - 이 게임은 `image_matching`의 핵심 확장 모드다

### 2. 자리 기억 게임
- 규칙:
  - 친구 또는 캐릭터가 어디 있었는지 기억해서 맞춘다
- 필요한 것:
  - 자리 배치판
  - 자리 데이터

### 3. 하루 일과 순서 맞추기
- 규칙:
  - 어린이집 일과를 순서대로 맞춘다
- 필요한 것:
  - 일과 카드 세트

### 4. 교실 규칙 선택 게임
- 규칙:
  - 상황을 보고 맞는 행동을 선택한다
- 필요한 것:
  - 상황 이미지
  - 선택지 카드

### 5. 이름과 초성 연결
- 규칙:
  - 이름 또는 단어와 초성을 매칭한다
- 필요한 것:
  - 한글 텍스트 카드

### 6. 반 친구 역할 맞추기
- 규칙:
  - 얼굴 또는 이름을 보고 사물함, 자리, 도우미 역할을 연결
- 필요한 것:
  - 반 정보 데이터

## Build Rule
- 학기 초기 적응 테마를 가장 강하게 넣는다
- 기록 저장과 개인정보 옵션이 중요하다
- 사진과 이름을 쓰는 기능은 반드시 `저장 안 함 기본값`을 유지한다

## First Build Priority
1. 친구 얼굴 이름 매칭
2. 자리 기억 게임
3. 하루 일과 순서 맞추기

## Needed Assets
- 얼굴 사진
- 이름 데이터
- 반 이름/관리자 정보
- 일과 카드
- 교실/자리 배치 이미지
