# Cleanup Timer Concepts

## 목적
놀이 후 정리 시간에 남은 시간을 시각적으로 보여주는 루틴 도구.
교사가 시작만 하면 자동 진행. 교실 TV/모니터/태블릿에서 사용.

## 공통 요건
- **시간 설정:** 프리셋(3/5/10/15분) + 직접 입력
- **화면:** 남은 시간 숫자 + 시각적으로 줄어드는 요소 + 정리 유도 문구
- **오디오:** 시작 효과음 · 1분 전/30초 알림 · 종료 효과음
- **추가:** 전체화면 · 음소거 · 배경 테마 변경

## 컨셉 3종

### Concept 1. Rainbow Countdown (추천 1순위)
- 큰 무지개가 시간 감소에 따라 색 띠가 하나씩 사라지거나 짧아짐
- 연령: 만 2~4세 / 적합 이유: 색 변화 직관적, 따뜻한 분위기
- 종료: 무지개 완전 사라짐 → `정리 끝` / `모여주세요`

### Concept 2. Character Friends Countdown (추천 2순위)
- 캐릭터 얼굴 N개(= 분 수)가 1분마다 하나씩 사라짐
- 연령: 만 2~5세 / 적합 이유: 개수 감소 직관적, 캐릭터 몰입감
- 종료: 마지막 캐릭터 사라짐 → 종료 메시지

### Concept 3. Clean-Up Rocket Gauge (추천 3순위)
- 로켓/기차가 시간 경과에 따라 목적지로 이동
- 연령: 만 4~5세 / 적합 이유: 목표 지향성, 이야기성
- 마지막 30초: 속도감 있는 연출 → 도착 → `모두 모여요`

## 릴리즈 계획

| 버전 | 내용 |
|------|------|
| v1 | Rainbow Countdown 단일 구현 + 기본 타이머 기능 |
| v2 | 3개 컨셉 선택 + 배경/테마 변경 + 정리/줄서기/손씻기 프리셋 |

## 결론
- 첫 구현: Rainbow Countdown (연령 범위 넓고, 직관적, 시각적으로 좋음)
- 시간 설정: 버튼식 프리셋 + 직접 입력 병행
- 게임과 분리된 `생활 루틴 도구`로 같은 사이트 내 제공
