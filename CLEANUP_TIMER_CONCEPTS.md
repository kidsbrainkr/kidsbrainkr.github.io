# Cleanup Timer Concepts

## Purpose
- 놀이 후 정리 시간에 아이들이 남은 시간을 직관적으로 이해하고 스스로 행동할 수 있도록 돕는 별도 화면 기획 문서다.
- 역할은 모두 동일하다:
  - 남은 시간을 시각적으로 보여준다
  - 시간이 줄어드는 과정을 아이가 눈으로 이해하게 한다
  - 정리, 줄 서기, 역할 수행 같은 행동을 스스로 하도록 유도한다

## Core Product Idea
- 이 화면은 일반 게임과 분리된 `생활 루틴 지원 도구`로 보는 것이 좋다
- 교실에서 TV, 모니터, 태블릿, 노트북으로 띄워둘 수 있어야 한다
- 조작은 교사가 시작할 때만 하면 되고, 실행 중에는 최대한 자동으로 진행되어야 한다

## Common Requirements
- 시간 설정 방식:
  - 버튼 프리셋: 3분, 5분, 10분, 15분
  - 직접 입력: 분 또는 초 단위
- 화면 표시:
  - 남은 시간 숫자
  - 시각적으로 줄어드는 요소
  - 정리 유도 문구
- 오디오:
  - 시작 효과음
  - 1분 전 또는 마지막 30초 알림
  - 종료 효과음
- 추천 추가 기능:
  - 전체화면 버튼
  - 음소거 버튼
  - 배경 테마 변경

## Concept 1. Rainbow Countdown

### Idea
- 화면 위 또는 중앙에 큰 무지개가 있다
- 시간이 줄어들수록 무지개 색 띠가 하나씩 사라지거나 짧아진다

### Why It Works
- 아이들이 색 변화에 쉽게 반응한다
- 남은 시간이 점점 줄어드는 것이 명확하다
- 정리시간에 어울리는 따뜻한 분위기를 만들기 좋다

### Screen Flow
- 시작 전:
  - 3분 / 5분 / 10분 / 15분 버튼
  - 시작 버튼
- 진행 중:
  - 중앙 큰 무지개
  - 아래에 `정리할 시간이에요`
  - 남은 시간 숫자
- 종료:
  - 무지개가 완전히 사라짐
  - `정리 끝` 또는 `모여주세요`

### Best Use
- 만 2세~4세
- 색과 형태에 반응이 좋은 반

## Concept 2. Character Friends Countdown

### Idea
- 화면에 캐릭터 얼굴 여러 개가 줄지어 있다
- 시간이 지나면 하나씩 사라진다
- 예:
  - 10분이면 얼굴 10개
  - 5분이면 얼굴 5개

### Why It Works
- 아이들이 개수 감소를 쉽게 이해한다
- 친구나 동물 캐릭터를 쓰면 몰입감이 높다
- 역할 수행 화면처럼도 보일 수 있다

### Screen Flow
- 시작 전:
  - 시간 프리셋 버튼
  - 캐릭터 테마 선택 가능
- 진행 중:
  - 큰 캐릭터 얼굴 줄
  - 하나가 사라질 때마다 효과음 또는 반짝임
  - `친구들이 집에 가고 있어요. 우리도 정리해요`
- 종료:
  - 마지막 캐릭터까지 사라짐
  - 종료 메시지 표시

### Best Use
- 만 2세~5세
- 시각적 개수 개념을 쓰기 좋은 반

## Concept 3. Clean-Up Rocket Gauge

### Idea
- 로켓이나 기차가 목적지로 이동한다
- 시간 경과에 따라 천천히 도착 지점에 가까워진다
- 남은 시간이 줄수록 게이지도 줄거나 이동이 진행된다

### Why It Works
- `정리 완료 지점으로 가는 중`이라는 목표 지향성이 생긴다
- 만 4세~5세에 특히 잘 맞는다
- 단순 감소형보다 이야기성이 있다

### Screen Flow
- 시작 전:
  - 시간 선택
  - 로켓/기차 테마 선택
- 진행 중:
  - 큰 배경과 이동 캐릭터
  - 남은 시간 숫자
  - 마지막 30초에는 속도감 있는 연출
- 종료:
  - 목적지 도착
  - `이제 정리 끝, 모두 모여요`

### Best Use
- 만 4세~5세
- 조금 더 집중력이 있는 연령대

## Recommended Product Structure

### Version 1
- 타이머 전용 단일 웹페이지
- 기능:
  - 3/5/10/15분 버튼
  - 직접 입력
  - 시작 / 일시정지 / 다시시작
  - 전체화면
  - 음소거
  - 컨셉 1개 우선 구현

### Version 2
- 3개 컨셉 중 선택 가능
- 배경/테마 변경
- 정리시간, 줄서기시간, 손씻기시간 같은 프리셋

## My Recommendation
- 첫 구현은 `Rainbow Countdown`이 가장 좋다
- 이유:
  - 연령 범위가 넓다
  - 의미가 직관적이다
  - 시각적으로 예쁘고 부담이 적다
- 두 번째 추천은 `Character Friends Countdown`
- 세 번째 추천은 `Clean-Up Rocket Gauge`

## Final Conclusion
- 이 화면은 별도 제품으로 충분히 가치가 있다
- 게임과는 목적이 다르지만 같은 사이트 안에서 `생활 루틴 도구`로 제공하면 좋다
- 시간 설정은 `버튼식 + 직접 입력`을 함께 두는 방식이 가장 실용적이다
