# Age 2 Game Plan

## Age Focus
- 만 2세반
- 핵심은 `직관성`, `큰 이미지`, `짧은 규칙`, `실패 부담 없음`

## UI Rules
- 버튼 수 최소화
- 글자보다 그림 우선
- 한 화면에서 선택지 2~4개 이하
- 카드나 오브젝트 크기를 크게 사용
- 결과 화면도 짧고 단순하게 구성

## Recommended Games

### 1. 색깔 찾기 게임
- 규칙:
  - 예시 색 하나를 보여주고 같은 색을 누른다
- 필요한 것:
  - 큰 색상 카드
  - 정답/오답 효과음
- `image_matching`에서 가져올 것:
  - 결과 화면
  - 효과음 구조

### 2. 같은 그림 고르기
- 규칙:
  - 위에 보이는 그림과 같은 그림을 아래에서 고른다
- 필요한 것:
  - 2~3개 선택지
  - 큰 이미지
- 구현 메모:
  - 카드 뒤집기 기능 불필요

### 3. 큰 것 작은 것 고르기
- 규칙:
  - 큰 사과, 작은 사과 중 큰 것만 누르기
- 필요한 것:
  - 같은 종류의 다른 크기 이미지
- 구현 메모:
  - 비교 개념 강화용

### 4. 같은 소리 찾기
- 규칙:
  - 소리를 듣고 같은 동물 또는 물건을 누른다
- 필요한 것:
  - 오디오 파일
  - 소리 재생 버튼
- 구현 메모:
  - 오디오 기반 게임의 시작점

## Build Rule
- 타이머, 랭킹, 저장 기능은 기본적으로 넣지 않는다
- 화면은 이미지가 많이 차지해야 한다
- 1차는 `index.html` 단일 파일로 충분

## First Build Priority
1. 색깔 찾기
2. 같은 그림 고르기

## Needed Assets
- 큰 PNG 또는 SVG 이미지
- 짧은 효과음 2~3개
- 고대비 색상 세트
