# Routine Tool Ideas

## Position
- 루틴 도구: 교사가 실행, 아이들이 화면 보며 행동
- 핵심 요건: 자동 진행 · 큰 화면 · 직관성 · 낮은 조작 부담

## Priority Tools (먼저 만들 5종)

| # | 도구명 | 연령 | 난이도 |
|---|--------|------|--------|
| 1 | 정리 시간 타이머 | 2~5세 | 낮음 |
| 2 | 하루 일과 순서판 | 3~5세 | 낮음~중간 |
| 3 | 손씻기 순서 + 타이머 | 2~5세 | 낮음 |
| 4 | 정리 순서 보드 | 3~5세 | 낮음 |
| 5 | 오늘의 도우미/역할 보드 | 4~5세 | 낮음~중간 |

**각 도구 공통 필요 기능:** 전체화면 · 음소거 · 시작/일시정지/재시작

### 1. 정리 시간 타이머
- 3/5/10/15분 프리셋 + 직접 입력

### 2. 하루 일과 순서판
- 일과 카드 등록 · 현재 단계 강조 · 다음 단계 미리보기

### 3. 손씻기 순서 + 타이머
- 단계별 이미지 + 20~30초 타이머

### 4. 정리 순서 보드
- 순서 카드 3~5개 · 완료 체크 표시

### 5. 오늘의 도우미/역할 보드
- 이름/사진 표시 · 역할 슬롯 · 랜덤 선택 또는 수동 지정

## Additional Tools (6~12)

| # | 도구명 | 연령 |
|---|--------|------|
| 6 | 감정 체크 보드 | 3~5세 |
| 7 | 진정하기 호흡 화면 | 3~5세 |
| 8 | 줄서기 준비 화면 | 3~5세 |
| 9 | 모두 함께 완료 게이지 | 4~5세 |
| 10 | 화장실 순서 보드 | 2~4세 |
| 11 | 가방 정리 체크판 | 4~5세 |
| 12 | 식사 준비 순서 화면 | 2~5세 |

## Design Principles
- 멀리서도 잘 보이는 화면 · 짧고 굵은 텍스트
- 아이콘/이미지 중심 · 자동 진행형 · 교사 조작 최소화

## Note
- 게임보다 반복 사용 빈도가 더 높을 수 있음
- 어린이집 현장에서 실제 활용성이 높은 별도 제품군 가능성 있음
