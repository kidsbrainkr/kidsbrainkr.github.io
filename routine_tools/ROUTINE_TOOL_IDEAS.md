# Routine Tool Ideas

## Purpose
- 이 문서는 `routine_tools` 카테고리에서 만들 수 있는 생활 루틴 지원 도구 아이디어를 정리한다.
- 목표는 게임과 달리 교실 운영, 자기조절, 시간 이해, 역할 수행을 돕는 도구를 구조화하는 것이다.

## Core Position
- 게임:
  - 아이가 직접 플레이
- 루틴 도구:
  - 교사가 실행하고 아이들이 화면을 보며 행동
- 따라서 루틴 도구는 `자동 진행`, `큰 화면`, `직관성`, `낮은 조작 부담`이 중요하다

## Priority Tools

### 1. 정리 시간 타이머
- 역할:
  - 놀이 후 정리 시간을 시각적으로 보여줌
- 추천 연령:
  - 만 2세~5세
- 활용 상황:
  - 자유놀이 후 정리
  - 교구 정리
  - 활동 전환
- 필요 기능:
  - 3/5/10/15분 버튼
  - 직접 입력
  - 시작/일시정지/다시시작
  - 전체화면
  - 음소거
- 구현 난이도:
  - 낮음

### 2. 하루 일과 순서판
- 역할:
  - 지금, 다음, 그다음 활동을 한눈에 보여줌
- 추천 연령:
  - 만 3세~5세
- 활용 상황:
  - 등원 후
  - 하루 활동 안내
  - 전환 시간
- 필요 기능:
  - 일과 카드 등록
  - 현재 단계 강조
  - 다음 단계 미리보기
- 구현 난이도:
  - 낮음~중간

### 3. 손씻기 순서 + 타이머
- 역할:
  - 손씻기 단계를 순서대로 보여주고 시간도 같이 안내
- 추천 연령:
  - 만 2세~5세
- 활용 상황:
  - 간식 전
  - 화장실 후
  - 실외활동 후
- 필요 기능:
  - 손씻기 단계 이미지
  - 20초 또는 30초 타이머
  - 단계별 애니메이션
- 구현 난이도:
  - 낮음

### 4. 정리 순서 보드
- 역할:
  - 정리를 어떤 순서로 해야 하는지 알려줌
- 추천 연령:
  - 만 3세~5세
- 활용 상황:
  - 장난감 정리
  - 미술도구 정리
  - 책상 정리
- 필요 기능:
  - 순서 카드 3~5개
  - 현재 단계 강조
  - 완료 체크 표시
- 구현 난이도:
  - 낮음

### 5. 오늘의 도우미/역할 보드
- 역할:
  - 오늘 맡은 역할을 큰 화면으로 보여줌
- 추천 연령:
  - 만 4세~5세
- 활용 상황:
  - 출석 도우미
  - 간식 도우미
  - 정리 도우미
- 필요 기능:
  - 이름 또는 사진 표시
  - 역할 슬롯
  - 랜덤 선택 또는 수동 지정
- 구현 난이도:
  - 낮음~중간

## Additional Useful Tools

### 6. 감정 체크 보드
- 역할:
  - 오늘 기분을 얼굴 아이콘으로 표현
- 추천 연령:
  - 만 3세~5세
- 활용 상황:
  - 등원 직후
  - 감정 조절 활동
- 필요 기능:
  - 표정 선택
  - 선택 결과 표시
- 구현 난이도:
  - 낮음

### 7. 진정하기 호흡 화면
- 역할:
  - 숨 들이마시기 / 내쉬기를 시각적으로 안내
- 추천 연령:
  - 만 3세~5세
- 활용 상황:
  - 흥분 진정
  - 쉬는 시간
- 필요 기능:
  - 커졌다 작아지는 도형
  - 호흡 타이밍 안내
- 구현 난이도:
  - 낮음

### 8. 줄서기 준비 화면
- 역할:
  - 줄서기 전에 해야 할 일을 순서로 안내
- 추천 연령:
  - 만 3세~5세
- 활용 상황:
  - 바깥놀이 전
  - 하원 준비
- 필요 기능:
  - 준비 단계 카드
  - 완료 순서 표시
- 구현 난이도:
  - 낮음

### 9. 모두 함께 완료 게이지
- 역할:
  - 반 전체가 정리나 준비를 마칠수록 게이지가 차는 화면
- 추천 연령:
  - 만 4세~5세
- 활용 상황:
  - 협동 정리
  - 모두 모이기
- 필요 기능:
  - 진행 게이지
  - 단계별 채움 애니메이션
- 구현 난이도:
  - 중간

### 10. 화장실 순서 보드
- 역할:
  - 화장실 이용 순서를 시각적으로 안내
- 추천 연령:
  - 만 2세~4세
- 활용 상황:
  - 생활 습관 교육
- 필요 기능:
  - 단계별 그림
  - 짧은 안내 문구
- 구현 난이도:
  - 낮음

### 11. 가방 정리 체크판
- 역할:
  - 물병, 알림장, 겉옷 등 준비물을 확인
- 추천 연령:
  - 만 4세~5세
- 활용 상황:
  - 하원 준비
  - 소풍 준비
- 필요 기능:
  - 체크 항목
  - 완료 표시
- 구현 난이도:
  - 낮음

### 12. 식사 준비 순서 화면
- 역할:
  - 손씻기, 자리 앉기, 앞치마, 감사 인사 등 식사 전 준비 단계 안내
- 추천 연령:
  - 만 2세~5세
- 활용 상황:
  - 점심/간식 전
- 필요 기능:
  - 순서 카드
  - 현재 단계 강조
- 구현 난이도:
  - 낮음

## Best First Set
- 먼저 만들 가치가 가장 높은 조합:
  1. 정리 시간 타이머
  2. 하루 일과 순서판
  3. 손씻기 순서 + 타이머
  4. 정리 순서 보드
  5. 오늘의 도우미/역할 보드

## Design Principles
- 화면은 멀리서도 잘 보여야 한다
- 텍스트는 짧고 굵게
- 아이콘과 이미지 중심
- 자동 진행형이 좋다
- 교사 조작은 최소화한다

## Final Note
- `routine_tools`는 게임보다 반복 사용 빈도가 더 높을 수 있다
- 잘 만들면 어린이집 현장에서 실제 활용성이 높은 별도 제품군이 될 수 있다
